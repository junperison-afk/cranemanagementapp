# クレーン管理アプリ（Crane Management App）

天井クレーンの点検業務を一元管理するWebシステム

## 概要

点検業者が案件登録から報告書共有・精算まで一元管理できるWebシステムです。

- **ドメイン**: manage.crane-mitsumori.com
- **主な利用デバイス**: PC（主利用）、タブレット・スマートフォンからもレスポンシブWebアプリでアクセス可能

## 主な機能

- 取引先管理
- 営業案件管理（見積複数版管理）
- 契約管理（契約書自動生成）
- プロジェクト管理（受注から自動生成）
- 機器台帳管理
- 点検記録管理（オフライン対応・OCR連携検討）
- 精算状況記録・完了処理

## 開発ルール

詳細は `.cursor/rules/rules.mdc` を参照してください。

主なルール：
- `docs/設計書.md` および `docs/UI設計.md` を常に参照し、これらの内容に沿ってコーディング方針を決定
- Git コミット／プッシュはユーザーの指示がある場合に限り実施
- コード内の命名・コメントは原則日本語を使用
- 設計書に記載のない仕様・UIを決定する場合は、Zoho CRM の設計・作りを参考に補完
